How to install

[Download](https://drive.google.com/folderview?id=13SnJHwJ8-4xHQ55Z25bO9cj8WyfhcJAO)

Your device must be rooted and have Magisk

Go to the Magisk app 

Enter the Apktool.Zip Module

Once done, reboot the phone.

Good luck
